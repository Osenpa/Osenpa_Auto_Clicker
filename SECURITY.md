# Security Policy

## Supported Releases

Security fixes are provided on a best-effort basis for the latest public source version in this repository.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by email:

- `osenpacom@gmail.com`

Include the following if possible:

- A short description of the issue
- Reproduction steps
- Impact assessment
- Screenshots or proof of concept if relevant

Please do not open a public issue for an unpatched security problem.
